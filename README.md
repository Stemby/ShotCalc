Dependencies
------------
You only need Python 3, NumPy, SciPy, matplotlib and Tkinter. On Debian/Ubuntu:

    $ sudo aptitude install python3-scipy python3-matplotlib python3-tk

Run on GNU/Linux
----------------
    $ ./shotcalc.py
